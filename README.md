# 🌌 Acerca de OlaPlayStudio

**OlaPlayStudio** es una aplicación Android diseñada para reproducir música online mediante APIs REST, ofreciendo una experiencia multimedia sencilla, rápida y optimizada para streaming de canciones.

La aplicación permite:

- 🎵 Escuchar música online
- 🔎 Buscar canciones favoritas
- 📥 Descargar música
- 📤 Compartir canciones
- ⚡ Reproducir audio en tiempo real
- ☁️ Consumir APIs REST dinámicamente

El proyecto está orientado al aprendizaje y práctica de:

- Android Development
- REST APIs
- Streaming multimedia
- Gestión de audio
- Networking Android
- Aplicaciones musicales
